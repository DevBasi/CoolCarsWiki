## 🌟 Введение

**CoolCars** — это продвинутая система транспортных средств для вашего сервера Minecraft. Забудьте о простых «лодках на льду» — наш плагин предлагает полноценный физический движок, кастомные модели и глубокую настройку каждой детали.

> [!IMPORTANT]
> **ПРОЕКТ ТЕПЕРЬ БЕСПЛАТЕН!** Мы открыли исходный код и функционал для всех желающих. Пользуйтесь, создавайте и делитесь!

---

## ✨ Основные возможности

*   **🏎️ Реалистичная физика:** Ускорение, дрифт, сцепление с дорогой и работа подвески.
*   **🛠️ Модульная система:** Создавайте свои машины через YAML-конфиги. Никакого кодинга!
*   **📦 CustomModelData:** Полная поддержка ваших 3D-моделей.
*   **⛽ Игровые механики:** Топливо, багажники, ключи, износ и ремонт.
*   **📱 Современный UI:** Красивые индикаторы в экшн-баре.

## 🚀 Быстрая установка

1. Скачайте последнюю версию из [Releases]().
2. Закиньте в `/plugins/`.
3. Перезагрузите сервер.
4. Введите `/car spawn volga`.
