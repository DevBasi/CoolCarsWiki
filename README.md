## О плагине

**CoolCars** — это комплексное решение для реализации транспортных систем на серверах Minecraft. Плагин ориентирован на глубокую настройку и производительность, позволяя создавать уникальные модели автомобилей с реалистичным поведением.

## Основные возможности

*   **Физический движок:** Собственная симуляция движения, включающая расчет ускорения, инерции, сцепления с поверхностью и работу подвески.
*   **Гибкая настройка:** Конфигурация через YAML-файлы позволяет привязывать 3D-модели (CustomModelData), настраивать уникальные звуки и технические характеристики для каждой модели.
*   **Игровые механики:** Встроенные системы топлива, прочности, ремонта, багажников и системы ключей для управления доступом.
*   **Интерфейс:** Отображение состояния автомобиля (скорость, топливо) в Action Bar и функциональные GUI-меню.
*   **Оптимизация:** Минимальное влияние на Tick Rate сервера даже при активном использовании большого количества транспорта.

## Установка

1. Разместите `CoolCars.jar` в папке `plugins`.
2. Перезапустите сервер для генерации файлов конфигурации.
3. Проверьте установку командой `/car spawn volga`.

## Команды

| Команда | Описание |
| :--- | :--- |
| `/car spawn <model>` | Призвать автомобиль выбранной модели. |
| `/car givekey <player> <model>` | Выдать ключ от автомобиля игроку. |
| `/car reload` | Перезагрузить конфигурацию плагина. |
| `/car list` | Посмотреть список доступных моделей. |
